# roku-fyne

i broke my roku remote so i made a new one with go and fyne 
## build instructions

### windows

install go if you haven't already.

install fyne:


```go get fyne.io/fyne/v2```

build with:


```go mod tidy && go build```

this will create a roku-fyne.exe executable in the same directory.
